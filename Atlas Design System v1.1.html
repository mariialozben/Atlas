# Atlas Design System # v1.1.0 > "Minimalist design foundation that speaks two languages > and adapts to any digital vision. > One system, endless projects." Version: 1.1.0 Status: Active — in development Supported locales: English (EN) · Ukrainian (UA) Theme: Dark-first (#1A1A1A base) · Light mode supported --- ## 0. Brand ### Name Atlas ### Logo - Text: "Atlas" - Font: Lato Bold - Colour: #E0E0E0 (Platinum) - No icon/mark — wordmark only - Minimum size: 18px rendered height - Clear space: 8px on all sides - Never recolour, never italicise ### Tagline "Minimalist design foundation that speaks two languages and adapts to any digital vision. One system, endless projects." Ukrainian: "Мінімалістична дизайн-система, що говорить двома мовами і адаптується до будь-якого цифрового проєкту. Одна система — нескінченні проєкти." ### Voice - Clear and precise — no jargon - Equally natural in English and Ukrainian - Confident but never loud --- ## 1. Colour Palette All named colours taken directly from the approved swatch. Named tokens use Atlas colour names. Token Name Hex Use --color-jet-black Jet Black #1A1A1A Page background (primary bg) --color-onyx Onyx #222526 Surface bg (cards, panels) --color-graphite Graphite #353A3E Elevated surface, nav bg --color-ash Ash #BFBFBF Highlights, hover states, muted icons, dividers --color-platinum Platinum #E0E0E0 Primary text, logo, headings ### Semantic / accent colours --color-blue-500 — #2E7DD6 Primary actions, CTAs, links --color-blue-300 — #60A8E8 Secondary, inactive states --color-blue-50 — #E8F4FE Focus tint, hover bg on light --color-success — #1D9E75 Confirmations, saved states --color-warning — #EF9F27 Alerts, attention --color-danger — #E24B4A Errors, destructive actions ### Highlight rule Highlights (hover backgrounds, active indicators, selected states, focus rings) use --color-ash (#BFBFBF) at reduced opacity (10–20%) on dark surfaces, or as a full-opacity border/accent stroke on interactive elements. ### Contrast guidance - --color-platinum on --color-jet-black: passes AAA (≥7:1) - --color-platinum on --color-onyx: passes AAA - --color-platinum on --color-graphite: passes AA (≥4.5:1) - --color-ash on --color-jet-black: passes AA - Blue-500 on --color-jet-black: passes AA for large text --- ## 2. Typography ### Fonts (Google Fonts, free, open licence) Headers: Lato Bold (weight 700) Script: Latin + Cyrillic (Ukrainian supported) Import: fonts.googleapis.com/css2?family=Lato:wght@700 Body text: Merriweather Regular (weight 400) + Bold (700) Script: Latin + Cyrillic Import: fonts.googleapis.com/css2?family=Merriweather:wght@400;700 Mono (optional, data tables / code blocks): JetBrains Mono 400 Import: fonts.googleapis.com/css2?family=JetBrains+Mono ### Type scale Token Size Font Weight Line-h Use --text-xs 12px Merriweather 400 1.5 Captions, labels, badges --text-sm 14px Merriweather 400 1.6 Secondary body, hints --text-base 16px Merriweather 400 1.7 Body copy --text-lg 18px Lato 700 1.5 Sub-headings --text-xl 22px Lato 700 1.3 Section headings --text-2xl 28px Lato 700 1.2 Page titles --text-3xl 36px Lato 700 1.1 Hero headings --text-hero 48px Lato 700 1.0 Main screen wordmark ### Rules - Lato Bold for all headings (h1–h4) and navigation labels - Merriweather for all body copy, descriptions, table content - Sentence case always — no ALL CAPS in UI - Minimum rendered font size: 12px --- ## 3. Internationalisation (i18n) Strategy: JSON locale files, lightweight context/store. Files: /locales/en.json · /locales/uk.json On load: read localStorage key "atlas-locale" default to 'en' if absent On toggle: update context, re-render, save to localStorage All component labels, placeholders, error messages, button text, and nav items must have EN + UA entries. "Atlas" (brand name) is never translated. --- ## 7. Theming ### Dark mode (primary — all pages default to this) --bg-page: #1A1A1A (Jet Black) --bg-surface: #222526 (Onyx) --bg-elevated: #353A3E (Graphite) --text-primary: #E0E0E0 (Platinum) --text-muted: #BFBFBF (Ash) --highlight: #BFBFBF (Ash — borders, accents, hovers) --border: #353A3E (Graphite) ### Light mode (secondary) --bg-page: #F5F5F4 --bg-surface: #FFFFFF --bg-elevated: #EDEDEB --text-primary: #1A1A1A --text-muted: #5F5E5A --highlight: #353A3E --border: rgba(0,0,0,0.10) --- ## 8. Usage Guidelines - Always use named tokens — never raw hex values in components - Background of every page: --bg-page (#1A1A1A) - Logo: always Lato Bold, always #E0E0E0 — never altered - Highlights and hover states: always #BFBFBF (Ash) - Primary interactive colour: #2E7DD6 (blue-500) - Maintain min 4.5:1 contrast for all body text - Lang switcher must appear on every screen - All user-facing strings must have EN + UA entries - Modals and wizards must trap focus; Escape to dismiss - Main screen is always the entry point — loads before any component or foundations page --- ## 9.
